RenovateлЎң нҢЁнӮӨм§Җ лІ„м „ кҙҖлҰ¬ мһҗлҸҷнҷ”н•ҳкё°

Created on July 11, 2023

2023 В  В· В  Renovate В  Automation В  GitHub В  DevOps

н”„лЎңм қнҠёлҘј кҙҖлҰ¬н•ҳлӢӨ ліҙл©ҙ npm outdated лӘ…л №м–ҙлҘј нҶөн•ҙ м„Өм№ҳлҗң лӘЁл“Ҳл“Өмқҳ мөңмӢ  лІ„м „мқ„ нҷ•мқён•ҳкі  м—…лҚ°мқҙнҠён•ҳлҠ” кіјм •мқҙ кҪӨ лІҲкұ°лЎңмҡё л•Ңк°Җ мһҲмҠөлӢҲлӢӨ. нҠ№нһҲ м—¬лҹ¬ н”„лЎңм қнҠёлҘј кҙҖлҰ¬н•ҳкұ°лӮҳ нҢҖ лӢЁмң„лЎң мһ‘м—…н•  л•ҢлҠ” мқҳмЎҙм„ұ кҙҖлҰ¬к°Җ мҶҢнҷҖн•ҙм§Җкё° мүҪмҠөлӢҲлӢӨ. мқҙлҹ° мғҒнҷ©м—җм„ң RenovateлҘј мӮ¬мҡ©н•ҳл©ҙ мқҳмЎҙм„ұ м—…лҚ°мқҙнҠё кіјм •мқ„ мһҗлҸҷнҷ”н•ҳм—¬ к°ңл°ң мғқмӮ°м„ұмқ„ лҶ’мқј мҲҳ мһҲмҠөлӢҲлӢӨ.

мқҙлІҲ нҸ¬мҠӨнҢ…м—җм„ңлҠ” Renovate GitHub Appмқ„ м„Өм№ҳн•ҳкі  м„Өм •н•ҳм—¬ нҢЁнӮӨм§Җ лІ„м „мқ„ мһҗлҸҷмңјлЎң кҙҖлҰ¬н•ҳлҠ” л°©лІ•мқ„ м•Ңм•„ліҙкІ мҠөлӢҲлӢӨ.

RenovateлһҖ?

RenovateлҠ” мҶҢн”„нҠёмӣЁм–ҙ н”„лЎңм қнҠёмқҳ мқҳмЎҙм„ұ м—…лҚ°мқҙнҠёлҘј мһҗлҸҷнҷ”н•ҙмЈјлҠ” лҸ„кө¬мһ…лӢҲлӢӨ. м •кё°м ҒмңјлЎң м ҖмһҘмҶҢлҘј мҠӨмә”н•ҳм—¬ кө¬лІ„м „ мқҳмЎҙм„ұмқ„ м°ҫм•„лӮҙкі , мһҗлҸҷмңјлЎң Pull Request(PR)лҘј мғқм„ұн•ҙ мӨҚлӢҲлӢӨ.

м„Өм • кіјм •

1. Renovate GitHub App м„Өм№ҳ

к°ҖмһҘ лЁјм Җ н•ҙм•ј н•  мқјмқҖ GitHub м ҖмһҘмҶҢм—җ Renovate GitHub Appмқ„ м„Өм№ҳн•ҳлҠ” кІғмһ…лӢҲлӢӨ.

  1. Renovate GitHub App нҺҳмқҙм§ҖлЎң мқҙлҸҷн•©лӢҲлӢӨ.
  2. Install лІ„нҠјмқ„ нҒҙлҰӯн•©лӢҲлӢӨ.
  3. RenovateлҘј м Ғмҡ©н•  кі„м •мқҙлӮҳ мЎ°м§Ғ(Organization)мқ„ м„ нғқн•©лӢҲлӢӨ.
  4. All repositoriesлҘј м„ нғқн•ҳкұ°лӮҳ Only select repositoriesлҘј м„ нғқн•ҳм—¬ Renovateк°Җ мһ‘лҸҷн•ҳкёё мӣҗн•ҳлҠ” лҰ¬нҸ¬м§ҖнҶ лҰ¬м—җ к¶Ңн•ңмқ„ л¶Җм—¬н•©лӢҲлӢӨ.

2. renovate.json м„Өм • м¶”к°Җ

GitHub App м„Өм№ҳ нӣ„, Renovate лҙҮмқҙ мһ‘лҸҷн•ҳкё° мӣҗн•ҳлҠ” лҰ¬нҸ¬м§ҖнҶ лҰ¬мқҳ лЈЁнҠё л””л үнҶ лҰ¬м—җ м„Өм • нҢҢмқјмқ„ м¶”к°Җн•ҙм•ј н•©лӢҲлӢӨ.

н”„лЎңм қнҠё лЈЁнҠём—җ renovate.json нҢҢмқјмқ„ мғқм„ұн•ҳкі  м„Өм •мқ„ м¶”к°Җн•©лӢҲлӢӨ. лӢЁмҲңнһҲ кё°ліё м„Өм •(config:base)л§Ң мӮ¬мҡ©н•  мҲҳлҸ„ мһҲм§Җл§Ң, м—…л¬ҙ нқҗлҰ„м—җ л°©н•ҙк°Җ лҗҳм§Җ м•ҠлҸ„лЎқ л§ӨмЈј мӣ”мҡ”мқј мҳӨм „ 9мӢңм—җ MinorмҷҖ Patch м—…лҚ°мқҙнҠёлҘј лӘЁм•„м„ң PRмқ„ мғқм„ұн•ҳлҸ„лЎқ м„Өм •н•ҙліҙкІ мҠөлӢҲлӢӨ.

{
  "$schema": "https://docs.renovatebot.com/renovate-schema.json",
  "extends": ["config:base"],
  "timezone": "Asia/Seoul",
  "packageRules": [
    {
      "matchUpdateTypes": ["minor", "patch"],
      "groupName": "all non-major dependencies",
      "schedule": ["before 9am on Monday"]
    }
  ]
}
  • $schema: м„Өм • нҢҢмқјмқҳ мң нҡЁм„ұмқ„ кІҖмӮ¬н•ҳкі  мһҗлҸҷ мҷ„м„ұмқ„ лҸ„мҷҖмӨҚлӢҲлӢӨ.
  • extends: Renovateмқҳ кё°ліё м„Өм •(config:base)мқ„ мғҒмҶҚл°ӣм•„ мӮ¬мҡ©н•©лӢҲлӢӨ.
    • м°ёкі : config:baseлҠ” config:recommendedмҷҖ лҸҷмқјн•ҳл©°, мөңк·јм—җлҠ” config:recommended мӮ¬мҡ©мқҙ к¶ҢмһҘлҗ©лӢҲлӢӨ.
    • лҚ” лӮҳм•„к°Җ config:best-practicesлҘј мғҒмҶҚл°ӣмңјл©ҙ Renovate нҢҖмқҙ м ңм•Ҳн•ҳлҠ” лӘЁлІ” мӮ¬лЎҖ м„Өм •л“Өмқҙ м Ғмҡ©лҗҳм–ҙ лҚ”мҡұ м•Ҳм „н•ҳкі  нҡЁмңЁм Ғмқё кҙҖлҰ¬к°Җ к°ҖлҠҘн•©лӢҲлӢӨ.
  • timezone: мҠӨмјҖмӨ„л§Ғмқҳ кё°мӨҖмқҙ лҗ  мӢңк°„лҢҖлҘј н•ңкөӯ мӢңк°„(Asia/Seoul)мңјлЎң м„Өм •н•©лӢҲлӢӨ.
  • packageRules: нҢЁнӮӨм§Җ м—…лҚ°мқҙнҠё к·ңм№ҷмқ„ м •мқҳн•©лӢҲлӢӨ.
    • matchUpdateTypes: Minor л°Ҹ Patch лІ„м „ м—…лҚ°мқҙнҠёл§Ң лҢҖмғҒмңјлЎң н•©лӢҲлӢӨ.
    • groupName: мқҙ к·ёлЈ№м—җ мҶҚн•ң м—…лҚ°мқҙнҠёл“Өмқ„ н•ҳлӮҳмқҳ PRлЎң л¬¶м–ҙм„ң мІҳлҰ¬н•©лӢҲлӢӨ (мҳҲ: вҖңall non-major dependenciesвҖқ).
    • schedule: л§ӨмЈј мӣ”мҡ”мқј мҳӨм „ 9мӢң мқҙм „м—җл§Ң н•ҙлӢ№ PRмқ„ мғқм„ұн•ҳлҸ„лЎқ мҠӨмјҖмӨ„мқ„ м§Җм •н•©лӢҲлӢӨ.

мқҙл ҮкІҢ м„Өм •н•ҳл©ҙ нҸүмқј м—…л¬ҙ мӢңк°„м—җлҠ” мһҰмқҖ м•ҢлҰјмңјлЎң л°©н•ҙл°ӣм§Җ м•Ҡкі , мЈјмҙҲм—җ н•ң лІҲм—җ мқҳмЎҙм„ұмқ„ м—…лҚ°мқҙнҠён•  мҲҳ мһҲм–ҙ нҡЁмңЁм Ғмһ…лӢҲлӢӨ.

Renovateмқҳ лҢҖм•Ҳ

Renovate мҷём—җлҸ„ мқҳмЎҙм„ұ кҙҖлҰ¬лҘј мһҗлҸҷнҷ”н•  мҲҳ мһҲлҠ” лӢӨлҘё лҸ„кө¬л“Өмқҙ мһҲмҠөлӢҲлӢӨ.

npm-check-updates

CI/CD нҢҢмқҙн”„лқјмқё м—°лҸҷ м—Ҷмқҙ лЎңм»¬ к°ңл°ң нҷҳкІҪм—җм„ң к°ҖлҒ” н•ң лІҲм”© мқҳмЎҙм„ұмқ„ мқјкҙ„ м—…лҚ°мқҙнҠён•ҳкі  мӢ¶мқ„ л•Ң мң мҡ©н•ң CLI лҸ„кө¬мһ…лӢҲлӢӨ.

  • м„Өм№ҳ: npm install -g npm-check-updates
  • мӮ¬мҡ©: ncu лӘ…л №м–ҙлЎң м—…лҚ°мқҙнҠё к°ҖлҠҘн•ң нҢЁнӮӨм§ҖлҘј нҷ•мқён•ҳкі , ncu -uлЎң package.jsonмқ„ м—…лҚ°мқҙнҠён•ң л’Ө npm installмқ„ мӢӨн–үн•©лӢҲлӢӨ.
  • мһҘм җ: м„Өм • нҢҢмқј м—Ҷмқҙ мҰүмӢң мӮ¬мҡ©н•  мҲҳ мһҲмңјл©°, н”„лЎңм қнҠё мҙҲкё° м„ёнҢ…мқҙлӮҳ лҢҖк·ңлӘЁ м—…лҚ°мқҙнҠёк°Җ н•„мҡ”н•  л•Ң л№ лҘҙкІҢ вҖңмӢ№ м—…лҚ°мқҙнҠёвҖқн•ҳкё° мўӢмҠөлӢҲлӢӨ.
  • лӢЁм җ: мһҗлҸҷнҷ”лҗң лҸ„кө¬к°Җ м•„лӢҲлҜҖлЎң к°ңл°ңмһҗк°Җ мЈјкё°м ҒмңјлЎң м§Ғм ‘ мӢӨн–үн•ҙм•ј н•©лӢҲлӢӨ.

GitHub Dependabot

GitHubм—җм„ң кё°ліём ҒмңјлЎң м ңкіөн•ҳлҠ” мқҳмЎҙм„ұ кҙҖлҰ¬ лҸ„кө¬мһ…лӢҲлӢӨ.

  • мһҘм җ: GitHubм—җ лӮҙмһҘлҗҳм–ҙ мһҲм–ҙ лі„лҸ„мқҳ м„Өм№ҳлӮҳ к°Җмһ… м—Ҷмқҙ м„Өм •(dependabot.yml)л§ҢмңјлЎң л°”лЎң мӮ¬мҡ©н•  мҲҳ мһҲмҠөлӢҲлӢӨ. ліҙм•Ҳ м·Ём•Ҫм җ(Security Advisory)м—җ лҢҖн•ң лҢҖмқ‘мқҙ л§Өмҡ° л№ лҰ…лӢҲлӢӨ.
  • лӢЁм җ: Renovateм—җ л№„н•ҙ м„Өм •мқҳ мң м—°м„ұмқҙ лӢӨмҶҢ л–Ём–ҙм§Ҳ мҲҳ мһҲмңјл©°, м—¬лҹ¬ нҢЁнӮӨм§ҖлҘј к·ёлЈ№нҷ”н•ҳм—¬ PRмқ„ ліҙлӮҙлҠ” кё°лҠҘ(Grouping)мқҙ мғҒлҢҖм ҒмңјлЎң м ңн•ңм Ғмқј мҲҳ мһҲмҠөлӢҲлӢӨ.

мқҙлҜё Dependabotмқҙ мһҲлҠ”лҚ° RenovateлҘј мҚЁм•ј н• к№Ңмҡ”?

л§ҺмқҖ н”„лЎңм қнҠём—җм„ң мқҙлҜё Dependabotмқ„ мӮ¬мҡ©н•ҳкі  мһҲмҠөлӢҲлӢӨ. к·ёлҹјм—җлҸ„ л¶Ҳкө¬н•ҳкі  RenovateлЎң м „нҷҳн•ҳкұ°лӮҳ м¶”к°Җн•ҳлҠ” мқҙмң лҠ” л¬ҙм—Үмқјк№Ңмҡ”?

  1. PR л…ёмқҙмҰҲ к°җмҶҢ (Grouping): DependabotмқҖ кё°ліём ҒмңјлЎң нҢЁнӮӨм§Җ н•ҳлӮҳлӢ№ н•ҳлӮҳмқҳ PRмқ„ мғқм„ұн•©лӢҲлӢӨ. мқҳмЎҙм„ұмқҙ л§ҺмқҖ н”„лЎңм қнҠём—җм„ңлҠ” м•„м№Ёл§ҲлӢӨ мҲҳмӢӯ к°ңмқҳ PR м•ҢлҰјмқ„ л°ӣмқ„ мҲҳ мһҲмҠөлӢҲлӢӨ. мқҙлҠ” нҢЁнӮӨм§Җлі„лЎң лҸ…лҰҪм Ғмқё н…ҢмҠӨнҠёмҷҖ лҰ¬л·°к°Җ к°ҖлҠҘн•ҳлӢӨлҠ” мһҘм җмқҙ мһҲм§Җл§Ң, кҙҖлҰ¬н•ҙм•ј н•  PRмқҙ л„Ҳл¬ҙ л§Һм•„м§ҖлҠ” лӢЁм җлҸ„ мһҲмҠөлӢҲлӢӨ. л°ҳл©ҙ RenovateлҠ” м•һм„  м„Өм • мҳҲмӢңмІҳлҹј кҙҖл Ёлҗң нҢЁнӮӨм§Җл“Өмқ„ н•ҳлӮҳмқҳ PRлЎң л¬¶кұ°лӮҳ, лӘЁл“  Non-major м—…лҚ°мқҙнҠёлҘј н•ҳлӮҳлЎң л¬¶лҠ” л“ұ к°•л Ҙн•ң к·ёлЈ№нҷ” кё°лҠҘмқ„ м ңкіөн•ҳм—¬ н”јлЎңлҸ„лҘј мӨ„м—¬мӨҚлӢҲлӢӨ. л¬јлЎ  л¬¶мқё м—…лҚ°мқҙнҠё мӨ‘ нҠ№м • нҢЁнӮӨм§Җл§Ң л¬ём ңк°Җ л°ңмғқн•ңлӢӨл©ҙ, лҢҖмӢңліҙл“ңм—җм„ң н•ҙлӢ№ нҢЁнӮӨм§Җл§Ң л¶„лҰ¬н•ҳкұ°лӮҳ ignoreDeps м„Өм •мңјлЎң м ңмҷён•ҳлҠ” л“ұмқҳ м¶”к°Җ мһ‘м—…мқҙ н•„мҡ”н•  мҲҳ мһҲмҠөлӢҲлӢӨ.
  2. Dependency Dashboard: RenovateлҠ” Dependency DashboardлқјлҠ” мқҙмҠҲлҘј мғқм„ұн•ҳм—¬ нҳ„мһ¬ м—…лҚ°мқҙнҠё лҢҖкё° мӨ‘мқё нҢЁнӮӨм§Җ, л¬ҙмӢңлҗң нҢЁнӮӨм§Җ, м—җлҹ¬ лЎңк·ё л“ұмқ„ н•ңлҲҲм—җ ліҙм—¬мӨҚлӢҲлӢӨ. м—¬кё°м„ң мІҙнҒ¬л°•мҠӨлҘј нҒҙлҰӯн•ҳм—¬ мҲҳлҸҷмңјлЎң PR мғқм„ұмқ„ мҡ”мІӯн•  мҲҳлҸ„ мһҲмҠөлӢҲлӢӨ.
  3. м •көҗн•ң м„Өм •кіј н”„лҰ¬м…Ӣ: RenovateлҠ” л§Өмҡ° мғҒм„ён•ң м„Өм •мқҙ к°ҖлҠҘн•ҳл©°, config:recommended к°ҷмқҖ лӢӨм–‘н•ң н”„лҰ¬м…Ӣмқ„ м ңкіөн•ҳм—¬ ліөмһЎн•ң мҡ”кө¬мӮ¬н•ӯ(мҳҲ: лӘЁл…ёл ҲнҸ¬ нҷҳкІҪ, нҠ№м • мӢңк°„лҢҖ л°°нҸ¬ л“ұ)мқ„ м¶©мЎұмӢңнӮӨкё°м—җ лҚ” мң лҰ¬н•©лӢҲлӢӨ.

кІ°лЎ м ҒмңјлЎң, DependabotмқҖ вҖңліҙм•Ҳ нҢЁм№ҳвҖқмҷҖ вҖңк°ңлі„ ліҖкІҪмӮ¬н•ӯмқҳ лӘ…нҷ•м„ұвҖқм—җ к°•м җмқҙ мһҲкі , RenovateлҠ” вҖңк°ңл°ң мғқмӮ°м„ұ(PR кҙҖлҰ¬)вҖқкіј вҖңмҡҙмҳҒмқҳ мң м—°м„ұвҖқм—җ к°•м җмқҙ мһҲмҠөлӢҲлӢӨ. л”°лқјм„ң лӢЁмҲңн•ң ліҙм•Ҳ м—…лҚ°мқҙнҠёл§Ң н•„мҡ”н•ҳлӢӨл©ҙ DependabotмңјлЎң м¶©л¶„н•ҳм§Җл§Ң, м Ғк·№м ҒмңјлЎң мөңмӢ  лІ„м „мқ„ мң м§Җн•ҳл©ҙм„ңлҸ„ PR кҙҖлҰ¬ л№„мҡ©мқ„ мӨ„мқҙкі  мӢ¶лӢӨл©ҙ Renovateк°Җ мўӢмқҖ м„ нғқмқҙ лҗ  кІғмһ…лӢҲлӢӨ.

л§Ҳм№ҳл©°

мқҙм ң Renovateк°Җ мЈјкё°м ҒмңјлЎң мқҳмЎҙм„ұмқ„ кІҖмӮ¬н•ҳкі  м—…лҚ°мқҙнҠёк°Җ н•„мҡ”н•ң нҢЁнӮӨм§Җк°Җ мһҲмңјл©ҙ мһҗлҸҷмңјлЎң PRмқ„ мғқм„ұн•ҙ мӨ„ кІғмһ…лӢҲлӢӨ. мқҙлҘј нҶөн•ҙ к°ңл°ңмһҗлҠ” л№„мҰҲлӢҲмҠӨ лЎңм§Ғ к°ңл°ңм—җ лҚ” м§‘мӨ‘н•  мҲҳ мһҲкІҢ лҗҳл©°, н•ӯмғҒ мөңмӢ  мғҒнғңмқҳ м•Ҳм „н•ң мқҳмЎҙм„ұмқ„ мң м§Җн•  мҲҳ мһҲмҠөлӢҲлӢӨ.
